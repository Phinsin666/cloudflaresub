# CloudflareSub

一个基于 **Cloudflare Workers**
的轻量级订阅生成工具，用于整理节点链接并生成多种客户端可用的订阅格式。

该项目适合个人部署、自用或学习 Cloudflare Workers 的基本开发方式。

------------------------------------------------------------------------

# 功能

-   支持输入多个节点链接
-   支持批量替换服务器地址为指定 IP 或域名
-   自动生成多个客户端订阅格式
-   支持二维码生成，方便移动端导入
-   支持基础去重处理
-   支持 Cloudflare Workers 一键部署

当前支持的订阅格式：

-   原始节点订阅
-   Clash 订阅
-   Surge 订阅

# 部署方式（暂未编写）
1. fork本仓库后打开cloudflare点击「Compute」
2. 左侧点击「Workers 和 Pages」后，右上角点击「创建应用程序」
3. 点击Continue with Github后选择本仓库，等待安装完成
4. 点击左侧「存储数据库」-Workers KV(KV命名空间：1.生成随机密钥2.降低订阅链接的长度3.降低二维码复杂度)-右上角Create Instance-名称随意-创建
5. 回到「Compute」点击左侧Workers 和 Pages，点击刚刚创建好的应用程序「...」-查看绑定-点击绑定-KV命名空间-添加绑定
6. 变量名称：SUB_STORE，KV 命名空间选择刚刚创建的「名称随意」-添加绑定
7. 点击上方「设置」-「变量和机密」的添加-类型选择密钥-变量名称：SUB_ACCESS_TOKEN，值填写一个复杂一点的随机数字字母(密码)-部署
8. 点击左上角「概述」-打开刚刚创建好的成品网址-尽情操作吧！
